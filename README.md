# Angular_Web_Application

This is an Event Search Web Appliaction which has various functionalities with venue details and Favorites tab.

- FrontEnd - HTML,CSS, JavaScript, TypeScript
- Backend - NodeJS

In this project we have used

- NodeJS to build API's

- API's used
  - Ticket Master API
  - Google Geocoding AP
  - IPInfo.io API
  - Ticketmaster APIs,
  - Spotify APIs,
  - Google Maps APIs
  - Facebook APIs
  - Twitter APIs

This application has been deployed in Google Cloud
